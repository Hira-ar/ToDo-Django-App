# ToDo-Django-App
A static todo app built with django, bootstrap, css, html and Sqlite database.

This **TODO Django App** is a comprehensive task management application built with Django, Bootstrap, CSS, and SQLite3. It includes robust user authentication (sign up, login, logout) and full CRUD (Create, Read, Update, Delete) functionality for managing tasks. The user-friendly interface, designed with Bootstrap, ensures intuitive experience, while Django handles secure data management with SQLite3.

## Features

- User Authentication (Sign Up, Login, Logout)
- CRUD Operations for Tasks (Create, Read, Update, Delete)
- Secure Data Management with Django and SQLite3
